COMMON WORKFLOWS


	Compile C Program:
nano program.c - Create/Edit source code
gcc program.c -o program - Compile into executable
chmod +x program - Make Executable for running
./program - Execute the compiled program


	Manage Files:
mkdir project - Create directory structure
cd project - Navigate to workspace
touch file1.txt file2.txt - Create project files
ls -la - Verify file creation and permissions
chmod 600 secret.txt - Secure sensitive files


	Monitor System:
ps aux - Check all running processes
top - Monitor real-time system resources
df -h - Analyze disk space utilization
free -h - Review memory usage statistics


	PIPES & REDIRECTION
ls > files.txt          # Save output to file
ls >> files.txt         # Append output to file
cat < input.txt         # Use file as input
ls | grep ".txt"        # Pipe: find text files
ls 2> errors.txt        # Save errors to file
ls &> output.txt        # Save ALL output to file
command1 && command2    # Run command2 only if command1 succeeds
command1 || command2    # Run command2 only if command1 fails
command1 ; command2     # Run command2 regardless
